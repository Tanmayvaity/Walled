# 🖼️ Walled

A wallpaper browsing and management app for Android with offline support, infinite scrolling, and a seamless user experience.

## ✨ Features

- **🔍 Browse Wallpapers** — Explore thousands of high-quality wallpapers with smooth infinite scrolling
- **❤️ Favorites** — Save your favorite wallpapers for quick access
- **📥 Download** — Download wallpapers to your device
- **🎨 Set Wallpaper** — Apply wallpapers directly to home screen or lock screen
